# ChatGPT 项目快速总览

## 当前最高目标

本项目的最高目标不是搭建知识库本身，也不是单纯批量生成内容。

当前最高目标是：

- 持续分析小红书店铺数据
- 持续产出能提升流量、点击、进店、订单的爆款笔记
- 所有系统建设都必须服务于订单越来越多、数据越来越好

判断系统是否有效，只看内容数据和成交数据是否持续变好。

## 当前主推产品

- 产品名称：班级通知
- 售价：17.9 元
- 成交情况：小红书店铺已成交 400 多单
- 目标用户：老师，尤其是班主任、任课老师、办公室老师
- 核心痛点：老师不用跑教室叫学生、叫班长、叫课代表

产品真实使用方式：

老师在办公室电脑或手机发送通知，教室电脑或一体机接收端直接弹出通知，学生抬头就能看到。

## 当前系统能力

- 数据复盘
- 笔记生成
- 做图交接
- 发布包归档
- info.json 生成
- GitHub 同步
- 人工修改学习
- 发布后复盘进化

## 当前发布包规则

每套发布包必须包含：

- 主图.png
- 副图1.png
- 副图2.png
- 副图3.png
- 副图4.png
- 副图5.png
- 发布文案.md
- info.json

`info.json` 的 `content` 必须使用 `\n\n` 保留段落空行。

发布助手优先读取 `info.json`，不要依赖 Markdown 浏览器预览里的复制效果。
