# 当前主推产品说明

## 当前主推产品

当前主推产品是：**班级通知**。

班级通知是一款教师办公效率软件，核心价值是让老师不用从办公室反复跑到教室叫学生、叫班长、叫课代表或传达临时通知。老师可以在办公室电脑或手机上发送通知，教室电脑/一体机接收端直接弹出通知，学生在教室就能看到。

## 历史项目说明

`期末评语生成助手` 是历史项目和参考样板，不是当前主推产品。

它可以继续作为以下内容的参考：

- 产品资料卡格式参考
- 教师痛点写法参考
- 小红书笔记结构参考
- 图片方案拆解参考

但当前内容测试、选题生成、爆款迁移、做图交接和笔记方案生成，都应优先围绕 **班级通知**。

## 当前测试优先级

当前所有测试任务优先围绕：

- 老师不用跑教室叫学生
- 老师在办公室一键通知教室
- 教室一体机弹出通知
- 班长拿试卷
- 课代表拿作业
- 临时通知某个学生或班级
- 班主任和任课老师的日常办公效率

不优先围绕：

- 家长通知
- 家长回执统计
- 家校沟通平台
- 成绩发布系统
- 期末评语生成

## 上游输入

当前内容系统上游接：**店铺数据分析窗口**。

店铺数据分析窗口负责把小红书店铺和笔记表现整理成结构化结论，包括：

- 当前数据口径
- 核心数据变化
- 高成交笔记
- 高播放但低转化笔记
- 低播放但高转化笔记
- 有效内容方向
- 低效内容方向
- 下一轮笔记生成指令
- 发布后观察指标

本知识库接收这些数据分析结果后，把它们转成可执行的内容生产任务。

## 下游输出

当前内容系统下游接：**做图 Skill 窗口**。

本知识库输出的做图交接单，需要让做图 Skill 能直接理解：

- 本篇笔记主题
- P1-P6 每张图的文案
- 每张图的画面要求
- 电子屏幕上的关键文字
- 整体风格
- 必须保留和不能改的文字
- 发布文案、标签和第一条评论

## 本地知识库的作用

本地知识库不是自动发布工具，也不是抓取工具。

它的核心作用是：

1. 接收“店铺数据分析窗口”的结构化结果。
2. 结合本地产品资料、用户痛点、创作规则、图片规则。
3. 生成完整的小红书笔记方案。
4. 生成适配做图 Skill 的做图交接单。
5. 把有效表达、低效表达和人工修改规律沉淀下来。

## 当前内容生产链路

店铺数据分析结果 -> 本地知识库理解和迁移 -> 完整小红书笔记方案 -> 做图交接单 -> 做图 Skill 出图 -> 人工检查 -> 人工发布。

第一阶段只做知识库和模板，不做自动发布、不做小红书抓取、不做自动出图、不接 MCP。
