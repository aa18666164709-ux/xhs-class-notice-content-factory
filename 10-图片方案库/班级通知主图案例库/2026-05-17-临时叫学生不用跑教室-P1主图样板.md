# 2026-05-17 临时叫学生不用跑教室 P1 主图样板

## 案例名称

临时叫学生 / 张同学来办公室 / 不用跑教室喊

## 对应产品

班级通知

## 对应场景

老师临时叫学生来办公室。

典型使用链路：

1. 老师在办公室电脑或手机上发送通知。
2. 教室电脑/一体机弹出通知。
3. 学生看到屏幕上的通知后，知道需要来办公室。
4. 老师不用从办公室跑到教室门口喊人。

## 最终主图画面结构

1. 主标题特别大，放在画面最容易被看到的位置，手机缩略图里也要能看清。
2. 画面同时出现办公室老师和教室大屏，让老师一眼看懂“办公室发送 -> 教室屏幕弹出”的使用链路。
3. 教室屏幕里的通知文字只保留一句，使用大字：张同学来办公室。
4. 画面整体保持真实校园感，像老师真实工作场景，不要做成科技感软件广告。
5. “班主任日常”可以作为小标签出现，但不能抢主标题。
6. 产品名“班级通知”可以放右下角小角标，不要作为主视觉。

## 为什么这个版本更好

1. 主标题直接写老师具体痛点，而不是抽象产品名。
2. “老师临时叫学生，不用再跑教室喊了”比“班级通知软件”更有点击感。
3. 老师看到主图后，能立刻理解这是不是自己的日常麻烦。
4. 办公室老师和教室大屏同时出现，降低理解成本。
5. 屏幕文案“张同学来办公室”短、具体、真实，比长句更适合做图，也更容易在缩略图里看清。
6. 真实校园感更强，不像泛泛的软件广告。

## 前几个版本的问题总结

1. 主标题不够大时，手机缩略图里看不清，点击诱因变弱。
2. 如果先突出“班级通知软件”，老师会先感受到产品感，而不是先感受到自己的痛点。
3. 如果画面只出现屏幕弹窗，没有办公室老师，使用链路不够完整。
4. 如果画面只出现老师，没有教室大屏，产品解决方式不够直观。
5. 如果教室屏幕文字太长，会降低识别效率，也容易生成错字。
6. 如果画面太科技感，会偏离教师真实日常，容易像软件广告。
7. 如果标签或产品名太大，会抢走主标题注意力。
