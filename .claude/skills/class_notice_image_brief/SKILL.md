# class_notice_image_brief

## 功能

把已生成的班级通知笔记方案整理成可直接复制给做图窗口的标准提示词。

本 Skill 不生成图片，不调用外部工具，只输出做图交接词。

## 触发语

- 生成做图交接词
- 生成 P1 做图提示词
- 生成 P2-P6 做图提示词
- 调用班级通知做图 Skill

## 重要要求

每次输出必须明确写：

**请调用 class-notice-xhs-note / 小红书做图工作流。**

## 输入来源

优先读取：

- `12-笔记生成结果/01-待检查/note编号_班级通知_主题/01-笔记方案.md`
- `12-笔记生成结果/01-待检查/note编号_班级通知_主题/02-做图交接单.md`
- `12-笔记生成结果/00-做图交接单模板.md`
- `09-创作规则库/班级通知内容生产总规则.md`
- `14-人工修改学习库/写作偏好档案.md`

## 输出分两种

### A. P1 主图单独生成版

用于先单独生成和打磨主图。

P1 输出必须包含：

- 调用哪个 Skill
- 产品名称
- 笔记主题
- 目标人群
- 核心痛点
- 主标题
- 副标题
- 小标签
- 电子屏幕文案
- 画面说明
- 排版要求
- 禁止事项

### B. P2-P6 副图批量生成版

用于批量生成副图。

P2-P6 每张图必须包含：

- 页码
- 标题
- 正文要点
- 电子屏幕文案
- 画面说明
- 排版要求
- 禁止事项

## 质量检查

输出前检查：

- 是否明确写了调用 `class-notice-xhs-note / 小红书做图工作流`。
- P1 是否可单独复制给做图窗口。
- P2-P6 是否可批量复制给做图窗口。
- 每张图是否都有电子屏幕文案。
- 电子屏幕文案是否具体到真实学校任务。
- 是否避免家长通知、回执统计和复杂后台。
