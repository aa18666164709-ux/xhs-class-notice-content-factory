# class_notice_image_brief

## 功能

把已生成的班级通知笔记方案整理成可直接复制给做图窗口的标准提示词。

本 Skill 不生成图片，不调用外部工具，只输出做图交接词。

默认输出“完整图片包提示词”，不是只输出 P1 主图提示词。

## 触发语

- 生成做图交接词
- 生成完整做图提示词
- 生成 P1 做图提示词
- 生成 P2-P6 做图提示词
- 调用班级通知做图 Skill

## 重要要求

每次输出必须明确写：

**请调用 class-notice-xhs-note / 小红书做图工作流。**

## 输入来源

优先读取：

- `12-笔记生成结果/01-待检查/note编号_班级通知_主题/01-笔记方案.md`
- `12-笔记生成结果/01-待检查/note编号_班级通知_主题/02-做图交接单.md`
- `12-笔记生成结果/00-做图交接单模板.md`
- `09-创作规则库/班级通知内容生产总规则.md`
- `14-人工修改学习库/写作偏好档案.md`

## 默认输出：完整图片包提示词

普通 6 篇测试、日常批量发布、快速测数据时，默认输出完整图片包提示词。

每套提示词必须要求做图窗口一次性生成：

- `主图.png`
- `副图1.png`
- `副图2.png`
- `副图3.png`
- `副图4.png`
- `副图5.png`

并保存到对应发布包文件夹。

完整图片包提示词必须包含：

### 一、主图提示词

- 主标题
- 副标题
- 小标签
- 电子屏幕文案
- 画面说明
- 排版要求
- 禁止事项

### 二、副图1-副图5提示词

每张包含：

- 标题
- 短文案
- 电子屏幕文案
- 画面说明
- 排版要求

### 三、保存要求

明确要求做图窗口生成后保存到对应发布包文件夹：

```text
主图.png
副图1.png
副图2.png
副图3.png
副图4.png
副图5.png
```

## 可选输出：分阶段提示词

### A. P1 主图单独生成版

仅在用户明确要求“先看主图”、主图跑偏、文字错误或本轮要重点打磨爆款主图模板时使用。

P1 输出必须包含：

- 调用哪个 Skill
- 产品名称
- 笔记主题
- 目标人群
- 核心痛点
- 主标题
- 副标题
- 小标签
- 电子屏幕文案
- 画面说明
- 排版要求
- 禁止事项

### B. P2-P6 副图批量生成版

仅在用户已经先生成主图、并明确要求补副图时使用。

P2-P6 每张图必须包含：

- 页码
- 标题
- 正文要点
- 电子屏幕文案
- 画面说明
- 排版要求
- 禁止事项

## 质量检查

输出前检查：

- 是否明确写了调用 `class-notice-xhs-note / 小红书做图工作流`。
- 默认是否输出完整图片包提示词。
- 是否明确保存为 `主图.png` 和 `副图1.png` 到 `副图5.png`。
- P1 是否包含主标题、副标题、小标签、电子屏幕文案、画面说明、排版要求和禁止事项。
- 副图1-副图5 是否每张包含标题、短文案、电子屏幕文案、画面说明和排版要求。
- 每张图是否都有电子屏幕文案。
- 电子屏幕文案是否具体到真实学校任务。
- 是否避免家长通知、回执统计和复杂后台。
