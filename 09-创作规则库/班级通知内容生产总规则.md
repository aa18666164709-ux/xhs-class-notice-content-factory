# 班级通知内容生产总规则

## 总原则

班级通知内容必须先写老师真实场景和痛点，再写产品解决方式。

核心表达：

老师在办公室电脑或手机发送通知，教室电脑/一体机弹出通知，学生能看到，老师不用为了叫学生、叫班长、叫课代表反复跑教室。

## 标题规则

标题必须优先包含以下至少两个元素：

- 老师身份：班主任、任课老师、办公室老师
- 具体任务：叫学生、叫班长、叫课代表、叫组长、拿试卷、拿作业
- 核心痛点：不用跑教室、不用传话、不被打断
- 功能结果：教室一体机弹出、大屏直接提醒、办公室发一句

推荐标题方向：

- 临时叫学生，不用再跑教室喊了
- 班长拿试卷，教室大屏直接提醒
- 课代表收作业本，办公室直接通知
- 在办公室发一句，教室屏幕直接弹出
- 老师每天最烦的小跑腿，可以少一点

标题避免：

- 班级通知系统
- 教师办公平台
- 学校通知工具
- 家校通知神器
- 全校必备工具
- 一键管理班级

## 正文规则

正文结构：

1. 开头写一个老师真实场景。
2. 写旧方法为什么麻烦。
3. 写产品怎么解决。
4. 写适合哪些具体场景。
5. 写使用边界和交付方式。
6. 轻转化，引导进店、评论或私信。

正文语气：

- 像老师之间分享经验。
- 具体、克制、真实。
- 不要像产品发布稿。
- 不要堆功能。
- 不要制造焦虑。

必须写清：

- 老师在办公室电脑或手机发送通知。
- 教室电脑/一体机弹出通知。
- 学生能看到。
- 店铺成交后提供飞书使用手册，包含发送端登录链接、接收端安装包、操作手册、激活卡号。

## P1 主图规则

P1 是成交意图筛选入口，必须一眼看出老师的具体麻烦。

P1 优先写：

- 老师不用跑教室叫人了
- 班长拿试卷，不用跑教室喊
- 课代表来拿作业，办公室直接通知
- 在办公室发一句，教室屏幕直接弹出
- 办公室离教室远的老师，真的懂

P1 画面必须突出：

- 办公室电脑或手机发送通知
- 教室电脑/一体机弹出通知
- 电子屏幕文案大、短、清楚

P1 不要写：

- 班级通知系统
- 教师办公平台
- 学校通知工具
- 家校通知神器
- 全校必备工具

## P2-P6 副图规则

推荐结构：

- P2：痛点场景，展示老师为什么要跑教室。
- P3：旧方法问题，展示跑、喊、传话的麻烦。
- P4：解决方式，展示办公室发送、教室屏幕弹出。
- P5：效果展示，展示学生看到通知并行动。
- P6：适合人群和转化，列出适合场景和交付方式。

每页必须有一个明确任务，不要只写泛泛功能。

适合屏幕文案：

- 班长来拿试卷
- 课代表送作业
- 张同学来办公室
- 各组长收作业
- 全班操场集合

## 标签规则

标签优先围绕教师人群和具体场景：

- #班主任
- #任课老师
- #老师日常
- #教师办公
- #班级通知
- #教室一体机
- #班级管理
- #教师效率
- #课代表
- #老师好物

避免标签：

- #家校沟通
- #家长通知
- #回执统计
- #学校管理系统
- #办公平台

## 首评规则

首评用于补充边界和承接，不要硬催下单。

推荐首评结构：

1. 说明它不是家长通知或回执统计。
2. 说明它解决的是老师在办公室通知教室的小跑腿问题。
3. 说明教室电脑或一体机能作为接收端会更适合。
4. 轻轻引导需要的老师进店或私信。

## 禁用表达

- 家校沟通神器
- 家长通知系统
- 自动回执统计
- 全校必备
- 一键管理班级
- 万能教师办公平台
- 保证省多少时间
- 用了再也不用进教室
- 彻底解决班级管理

## 自检规则

每条班级通知笔记生成后，必须检查：

- 标题是否有具体老师任务。
- 主图是否先写痛点，而不是先写产品名。
- 正文是否先写场景，再写产品。
- 是否明确“不用跑教室叫人”。
- 是否明确“办公室发送，教室一体机弹出”。
- 是否没有写成家长通知、回执统计或家校沟通。
- P1-P6 是否都有电子屏幕文案。
- 电子屏幕文案是否短、大、清楚。
- 标签是否精准。
- 首评是否补充边界。
- 是否说明交付方式。

## 通过标准

一条合格笔记应该让老师在 3 秒内看懂：

“这就是我每天叫学生、叫班长、叫课代表时遇到的小麻烦。这个工具不是复杂系统，就是让我少跑几趟教室。”
