# 班级通知专项写作规则

## 产品核心表达原则

班级通知不是复杂软件，不是家长通知系统，也不是学校 OA 平台。

它的核心表达是：

老师在办公室电脑或手机发通知，教室电脑/一体机直接弹出，学生能看到。老师不用为了叫学生、叫班长、叫课代表、传达临时小事，反复从办公室跑到教室。

写作时必须先写老师真实场景和痛点，再写产品解决方式。

推荐表达顺序：

1. 老师正在办公室处理事情。
2. 突然需要通知教室里的某个人或全班。
3. 旧方法是跑教室、找人带话、走廊喊人。
4. 这件小事一天多次发生，很浪费时间。
5. 班级通知的做法是：办公室发，教室屏幕弹。
6. 最后自然说明购买和交付方式。

## 标题写法规则

标题优先写场景和痛点，不要一上来写“软件/系统/平台”。

### 推荐标题元素

- 老师不用跑教室
- 叫学生来办公室
- 叫班长拿试卷
- 叫课代表拿作业
- 办公室发通知
- 教室一体机弹出
- 班主任日常工具
- 17.9 元

### 推荐公式

- 场景 + 痛点：叫班长拿试卷，真的不用再跑一趟教室
- 人群 + 高频小事：班主任每天最烦的小跑腿，可以少一点
- 旧方法 + 新方法：以前跑教室喊人，现在办公室直接发到屏幕
- 价格 + 具体价值：17.9 元，解决老师反复跑教室的小麻烦

### 避免标题

- 班级通知系统上线了
- 教师办公平台推荐
- 学校数字化管理神器
- 家校通知回执工具

## 主图文案禁忌与推荐

### 主图不要写

- 班级通知系统
- 教师办公平台
- 学校通知工具
- 一键管理班级
- 家校通知神器
- 全校必备工具

### 主图优先写

- 老师不用跑教室叫人了
- 在办公室发一句，教室屏幕直接弹出
- 叫班长拿试卷，不用再跑一趟
- 课代表来拿作业，办公室直接通知
- 办公室离教室远的老师，真的懂

### 原则

主图必须先让老师感受到“这就是我的日常麻烦”，不要先展示产品名或软件感。

## 正文写法规则

正文要像老师之间分享一个好用的小办法，不要像产品发布稿。

### 推荐结构

1. 开头写具体瞬间：老师正在办公室，突然要叫学生或班长。
2. 写旧方法的麻烦：跑一趟、找人传话、打断手头工作。
3. 写产品使用方式：办公室电脑/手机发送，教室一体机弹出。
4. 写适合场景：叫学生、班长、课代表、临时通知。
5. 写交付方式：店铺拍下后给飞书手册，里面有链接、安装包、操作手册、激活卡号。
6. 轻转化：需要的话可以去店铺看，或评论/私信关键词。

### 正文语气

- 具体
- 克制
- 像真实老师日常
- 少用大词
- 少用感叹号
- 不制造焦虑

### 必须避免

- 不要把它写成家长通知或回执统计。
- 不要说它能解决所有班级管理问题。
- 不要把安装和使用说得过于复杂。
- 不要承诺“再也不用跑教室”。

## 标签写法规则

标签优先使用教师人群、真实场景和搜索关键词。

### 推荐标签

## 爆款结构适配规则

班级通知适合迁移“痛点解决型”“旧方法对比型”“小工具种草型”“真实场景演示型”“低价高频痛点型”。

迁移时只迁移结构，不照抄其他爆款的产品话术。

### 适配判断

- 是否能在 P1 里一眼看出“不用跑教室叫人”。
- 是否能在 P2-P3 里讲清旧方法麻烦。
- 是否能在 P4-P5 里讲清办公室发送、教室屏幕弹出。
- 是否能在 P6 里自然引导去店铺或私信。

2. 老师在办公室发一句，教室屏幕直接弹出来
3. 班主任叫学生来办公室，可以不用找人传话
4. 办公室离教室远的老师，会懂这个小工具
5. 课代表来拿作业这件事，我不想再跑一趟

## 5 类适合的爆款结构

### 1. 痛点解决型

办公室要叫学生 -> 跑教室很麻烦 -> 班级通知直接弹到教室屏幕 -> 适合高频通知场景。

### 2. 旧方法对比型

旧方法：跑教室、找同学传话、走廊喊人。

新方法：办公室电脑/手机发送，教室一体机弹出。

### 3. 小工具种草型

不讲大系统，只讲一个小功能：老师不用为叫人反复跑教室。

### 4. 场景清单型

列出适合使用的 5 个场景：叫学生、叫班长、叫课代表、通知班级、临时提醒。

### 5. 低价高频痛点型

17.9 元 + 老师每天都可能遇到的小跑腿问题 + 小红书店铺已成交 400 多单。

## 低效表达黑名单

| 低效表达 | 问题 | 替代表达 |
|---|---|---|
| 班级通知系统 | 太像复杂软件 | 老师在办公室发通知，教室屏幕弹出来 |
| 家校沟通神器 | 写偏产品 | 叫学生、叫班长、叫课代表不用跑教室 |

| 全校必备 | 夸大 | 适合办公室离教室远、经常叫学生的老师 |
| 用了再也不用跑教室 | 绝对化 | 能少跑一些没必要的来回 |
| 一键解决班级管理 | 夸大且偏题 | 解决通知教室这件小事 |
| 家长自动回执 | 产品不符 | 教室端弹出通知给学生看 |

## 高效表达示例

- “老师最烦的不是大事，是这种一天好几次的小跑腿。”
- “叫班长拿试卷，真的没必要每次都从办公室跑过去。”
- “它解决的不是复杂管理，就是办公室通知教室这件小事。”
- “电脑或手机发一句，教室一体机直接弹出来。”
- “适合办公室离教室远、经常要叫学生的老师。”
- “拍下后给飞书手册，里面有发送端链接、接收端安装包、操作手册和激活卡号。”

## AI 自检清单

生成班级通知相关内容后，必须逐条检查：

- 是否先写老师真实场景，再写产品。
- 是否明确表达“不用跑教室叫人”。
- 是否写清“办公室电脑/手机发送，教室电脑/一体机弹出”。
- 是否避免写成家长通知、回执统计或家校沟通。
- 是否避免一上来写“软件/系统/平台”。
- 是否没有夸大成万能班级管理工具。
- 是否至少包含一个具体场景：叫学生、叫班长、叫课代表、拿试卷、拿作业。
- 是否能自然承接小红书店铺成交。
- 是否说明交付方式是飞书使用手册。
- 是否保留价格 17.9 元和成交 400 多单时的克制表达。
- P1 是否突出“不用跑教室叫人”。
- 图片中的电子屏幕文案是否短、大、清楚。
